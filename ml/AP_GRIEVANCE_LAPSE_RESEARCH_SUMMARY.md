## Why So Few Cases? Data Availability Analysis

### Public Domain Limitations

1. **Portal Access Restrictions**
   - pgrs.ap.gov.in requires login for case details
   - Only aggregate dashboard statistics visible publicly
   - Individual case tracking needs grievance ID

2. **Privacy Protections**
   - Individual grievance details protected
   - Anonymization requirements limit public disclosure
   - RTI requests needed for detailed data

3. **CAG Audit Scope**
   - Focus on financial audits, not grievance quality
   - No dedicated PGRS performance audit found
   - MGNREGA social audits more detailed but limited scope

4. **News Coverage Pattern**
   - Reports systemic issues, not individual cases
   - High-profile cases only occasionally detailed
   - Most coverage is statistical/aggregate

5. **Academic Research Focus**
   - System design and adoption studies
   - Not focused on individual lapse cases
   - Qualitative issues vs quantitative case data
